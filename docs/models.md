# Supported Models Matrix

This document lists the official models available on the WeizeRouter network.

## Available Models

- `gpt-4o`: General-purpose flagship OpenAI model.
- `claude-3-5-sonnet`: Highest intelligence tier, optimized for reasoning.
- `gemini-1.5-flash`: Fast, low-latency, and cost-efficient.
- `llama-3.1-70b-instruct`: Open-weights champion for structured tasks.
