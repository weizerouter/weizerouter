## Package Activation

**Activation Time:** 5-10 seconds after payment confirmation

**Expiry Time:**
- Mini Trial: 2 hours from activation
- Trial: 24 hours from activation
- Pro: 24 hours from activation
- Ultimate: 24 hours from activation

**Renewal:** You can purchase a new package at any time. If purchased before expiry, the new package duration is added to your remaining time.

---

## Model Availability

**Important:** Model availability depends on:
- Your active package tier
- Gateway provider availability
- Fair use cooldown status

If a model is temporarily unavailable, the API will return an error. Check your dashboard for current status or try a different model.
