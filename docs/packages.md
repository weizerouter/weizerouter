# Packages

WeizeRouter provides several package tiers.

## Mini Trial

Short testing package for quick API access.

Best for:

- first-time users
- quick API check
- testing payment and activation flow

## Trial

Basic package for API testing.

Best for:

- simple testing
- short-term access
- checking compatibility

## Pro

Daily package for regular usage.

Best for:

- normal API usage
- daily workflow
- users who need stable access

## Ultimate

All Models access with Unlimited Fair Use.

Best for:

- coding
- advanced usage
- premium model access
- heavier workloads

## Notes

Model availability may change depending on:

- provider stability
- gateway configuration
- package policy
- fair-use protection
