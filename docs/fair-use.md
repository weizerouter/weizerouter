# Fair Use
WeizeRouter uses Fair Use protection to keep the service stable for all users.

## Why Fair Use Exists

Fair Use helps prevent:

- API abuse
- spam loops
- excessive automated requests
- unstable provider usage
- service disruption for other users

## Cooldown System

When usage is extremely high, the system may activate temporary cooldown.

Example message:

> Cooldown Sementara  
> Penggunaan API kamu sedang sangat tinggi, jadi sistem Fair Use aktif untuk menjaga performa tetap stabil.  
> Silakan tunggu sebelum mencoba lagi.

## Goal

The goal is not to limit normal users.

The goal is to keep the gateway stable and fair for everyone.
